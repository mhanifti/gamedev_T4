## Level 1

> Rintangan Jurang, FishFall, dan Tanjakan. Dimana ketika menyentuh Jurang, maka akan respawn, dan ketika mengenai FishFall akan kalah (LoseCondition/LoseScreen). Ketika mengenai rocket, maka akan WinNextLevel (Bare Minimum)

> Rintangan Jurang, WormFall. Dimana ketika menyentuh Jurang, maka akan respawn, dan ketika mengenai WormFall akan kalah (LoseCondition/LoseScreen). Ketika mengenai rocket, maka akan WinCondition. (Bare Minimum)
